Comments on the Git audit trail activity

The Git audit trail does not accurately reflect the team’s activity. By the end of A2, majority of the team did not use Git to upload their work. Majority of the git commits made on the repo in the earlier weeks were from Bo. There are periods of time midway through A2 where there were no git commits whatsoever. It is only when we reach the A2 deadline that we saw commits being made again, only for the website’s HTML/CSS document. Again, not all members made these commits. Based on overall sentiment, the team preferred to use Microsoft Teams to upload their files and reports as opposed to GitHub. Therefore, it would be better to look at the Microsoft Team’s Group log of activity to have a better idea on the group’s progress for A2. There were multiple attempts to push for the use of Git for uploading files/reports, but they were unsuccessful.
